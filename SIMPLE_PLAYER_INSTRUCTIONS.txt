### STEP 1: Create Your Playlist

Create a text file called `playlist.txt` with your video URLs (one per line):

```
https://example.com/video1.mp4
C:\Videos\movie1.mp4
https://example.com/video2.mp4
C:\Videos\movie2.mp4
```

OR just copy/paste URLs directly into the HTML file (see Step 3)

### STEP 2: Add Thumbnails (Optional)

1. Create a folder called `thumbs/` next to your HTML file
2. Add JPG images with same name as your videos:
   - video1.jpg
   - movie1.jpg
   - video2.jpg
   - movie2.jpg

If no thumbnail found, shows a default play icon.

### STEP 3: Generate Player Page

**Option A: Use M3U Matrix**
1. Open M3U Matrix
2. Click "Generate NEXUS TV Page"
3. Choose "Simple Player" template
4. Paste your URL list
5. Generate!

**Option B: Manual**
1. Open `templates/simple_nexus_player.html` in Notepad
2. Find this line:
   ```javascript
   const playlistText = `__PLAYLIST_DATA__`;
   ```
3. Replace `__PLAYLIST_DATA__` with your URLs (one per line)
4. Save file

### STEP 4: Start Server and Play

1. Run: `python -m http.server 8000`
2. Open: `http://localhost:8000/simple_nexus_player.html`
3. Videos play automatically!
